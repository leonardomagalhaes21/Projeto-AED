Adds an UC to a student's schedule.

@detail This function adds a UC to a student's schedule if certain conditions are met. The conditions include checking the student's UC count, ensuring there are no schedule conflicts, and checking the UC's capacity.

Parameters

mThe map of UCs to students.
Returns
